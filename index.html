CSS Snippets

Snippets of CSS used to make this page! Simply click a code area to highlight all of it.

Root

Sets variables for the colors used

:root{
    --primary: #771177;
    --secondary: #241177;
    --primary-text: white;
}

Header

Gives header a gradient and centers text

header{
    display: block;
    text-align: center;
    background: linear-gradient(var(--primary), var(--secondary));
    color: var(--primary-text);
    padding: 20px 30px;
}

Cards

CSS used to make these cards

main section{
    width: 30%;
    border: black solid 3px;
    margin: 10px 0px;
    padding: 1px;
    background-color: gray;
    transition: width 1s ease-in-out;
}

Hover behavior

Cards widen when the cursor is hovered over them

main section:hover{
    width: 35%;
}

Smaller screens

Less cards per row when on a smaller screen

@media screen and (max-width: 992px){
    main section{
        width: 45%;
    }
}

@media screen and (max-width: 768px){
    main section{
        width: 90%;
    }
}
